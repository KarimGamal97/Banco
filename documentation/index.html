Installation

To install Bankly on your web server, download the files from Themeforest and unzip them to the local disk on your computer. Inside the extracted folder you will see three folders named Bankly - Documentation, Bankly - Template. Bankly - Template folder structure is as follows:

  • assets - contains all assets of this template
    • css - contains all main style css files
    • imgs - contains all images used in the template
    • js - contains all vendor and custom js files
    • vendor - contains all plugins and libraries used in the tepmlate
  • index.html - Home Page

HTML Structure

This template features a responsive, full-screen layout and is built using modern HTML5 standards.

index.html

From beginning of the <body> tag till the ending </body> tag, The code and text in the template define its structure. To update any text, simply open the index.html file and find the specific lines you want to edit.

Like index.html, you can change any page included with this template starting from <body> tag till ending </body> tag.

CSS Files

All the css files are located in the css folder of the template.

You can edit style easily, add your style bottom of the style.css and see the changes

JavaScript Files

All vendor, 3rd party plugin and our custom JavaScript files are located in the js folder.
main.js - Our custom JavaScript file. You can add your custom js codes there if you need to.

Please DO NOT edit this files unless you have JavaScript coding experience. Simple mistakes can cause serious damage to the scripts
